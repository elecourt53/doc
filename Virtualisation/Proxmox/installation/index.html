    Téléchargement

    Il faut le télécharger gratuitement sur le site officiel.

    Création d'une clé bootable pour l'installation

    Il faut utiliser le logiciel Rufus qui permet de créer une clé USB d'installation grâce au fichier .iso téléchargé avant.
